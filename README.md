# BlackBelt — Visão Geral do Sistema

Sistema de gestão para academias e escolas marciais.

## Sobre o projeto

O **BlackBelt** é um sistema de gestão para academias e escolas marciais, projetado para centralizar e organizar processos administrativos, operacionais e de acesso da instituição.

## Stack
- Banco de dados: SQL Server
- Backend: ASP.NET Core 
- Frontend: React + TypeScript (Vite)

## Estrutura do projeto

- `/database` → scripts e versionamento do banco
- `/application` → backend (API)
- `/apps/web` → frontend
